# Technology Stack Matrix

Here are the technologies used in the DecisionFlow AI project:

## Frontend System
* **React 18**: View layer structure.
* **TypeScript**: Core language validation.
* **Vite**: Ultra-fast assets bundler.
* **Lucide React**: Vector icons engine.
* **Axios**: HTTP connection client.

## Backend System
* **FastAPI**: Modern, high-performance web API framework.
* **Python 3.10+**: Runtime programming language.
* **SQLAlchemy**: Core SQL Object-Relational Mapper.
* **Pydantic**: Data schema models.
* **Uvicorn**: Lightweight ASGI server.

## AI & Integrations
* **Gemini API**: Core LLM engine executing sentiment classifications and timeline summaries.
* **Gmail IMAP App Passwords**: Real-time support mailbox integration.

# B2B Customer Portal Module

Provides a dashboard for client employees to log telemetry logs, raise tickets, submit feedback, and view announcements.

* **Attendance Logging**: Simulates daily shift check-ins / check-outs, writing records into telemetry tables.
* **Support Ticket Creation**: Submits categorization logs directly to CSM queue.
* **CSAT & Sentiment Feedback**: Rates support resolution speed to trigger updates in client health scores.
